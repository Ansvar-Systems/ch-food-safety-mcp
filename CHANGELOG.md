# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-05

### Added

- Initial release with 10 tools (3 meta + 7 domain)
- Selbstkontrolle/HACCP requirements by business type (LMG Art. 26, HyV Art. 4-8)
- BLV vs. cantonal registration requirements
- Mandatory labelling rules with allergen declaration and Swissness origin marking
- Temperature requirements per food category (HyV Anhang)
- Direktvermarktung rules for Hofladen, Ab-Hof-Verkauf, Wochenmarkt
- AOC/AOP and IGP protected designations register
- FTS5 full-text search with tiered fallback (phrase, AND, prefix, stemmed, OR, LIKE)
- Streamable HTTP transport for remote access
- Docker container with health checks
- CI/CD: CodeQL, Gitleaks, GHCR build, npm publish, data ingestion, freshness checks
